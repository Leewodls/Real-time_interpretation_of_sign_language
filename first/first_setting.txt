 3.7 둘 이상의 사물을 같은 자격으로 이어주는 말(접속 조사 '며, 이며')

 4. 인용
4.1 앞말이 직접 인용되는 말임을 나타내는 말(부사격 조사 ‘라고, 라, 이라고, 이라’)

 ex4) 문(門) + 곳 + 입춘대길 + 쓰다 + 있다 + 끝 = 문에는 ‘입춘대길’이라 쓰여 있었다.

 ex6) 너 + 도 + 잘못 + /동작을 강하게 하며 강조하는 표정 + 있다 = 너에게도 잘못은 있다.
 ex7) 너 + 먹다 + 싶다 + 대로 + 마음껏 + /과장하는 동작과 표정 + 마시다 + /시키는 표정 = 네

 ex1) 이것 + 문제 + 관하다 + 보고서 + 작성하다 + 오다 + 부탁하다 = 이 문제에 관한 보고서를

 8.6 앞말이 기준이 되는 대상이나 단위임을 나타내는 말(부사격 조사 ‘에’)

 9.3 비교의 기준이 되는 대상임을 나타내는 말(부사격 조사 ‘랑, 이랑’)

 9.4 다른 것과 비교하거나 기준으로 삼는 대상임을 나타내는 말(격 조사 ‘과, 와, 하고’)

과는 옷차림이 다르다.

9.5 비교의 대상이 되는 부사어임을 나타내는 말(격 조사 ‘에’)

 9.6 비교의 기준이 되는 점의 뜻을 나타내는 말(부사격 조사 ‘보다, 에서’)
 ex1) 그 + 누가 + 보다 + 걸음 + 빠르다 = 그는 누구보다도 걸음이 빠르다.

있겠어?
 10. 선택 ‧ 나열 ‧ 열거
10.1 여러 가지 중에서 어느 것을 선택하여도 상관없음을 나타내는 말(보조사 ‘나, 이나’)

 ex2) 그 + 문학 + 또 + 음악 + 모두 + 소질 + 있다 = 그는 문학이나 음악이나 모두 소질이 있다.
 10.2 마음에 차지 아니하는 선택, 최소한 허용되어야 할 선택이라는 뜻을 나타내는 말(보조사 ‘나,

 10.3 가정된 가장 좋은 선택이라는 뜻을 나타내는 말(보조사 ‘나, 이나’)

나.
 ex2) 자기 + 무슨 + 대장 + 되다 + 처럼 + 굴다 = 자기가 무슨 대장이나 되는 것처럼 굴더라.
 10.4 어느 것이 선택되어도 차이가 없는 둘 이상의 일을 나열함을 나타내는 말(보조사 ‘거나’)
 ex1) 그녀 + 예쁘다 + 그리고 + 귀엽다 + 칭찬 + 매우 + 좋아하다 = 그녀는 예쁘다거나 귀엽다거
나 하는 칭찬을 매우 좋아한다.
 10.5 어느 것이 선택되어도 차이가 없는 둘 이상의 일을 나열함을 나타내는 말(보조사 ‘든, 든가, 
든지’)

 ex3) 사과 + 든지 + 배 + 든지 + 다 + 좋다 = 사과든지 배든지 다 좋다.
 10.6 어느 것이 선택되어도 차이가 없는 둘 이상의 일을 나열함을 나타내는 말(보조사 ‘이든, 이든
지’)

10.7 어느 것이 선택되어도 상관없는 사물들을 열거할 때 쓰는 말(보조사 ‘라든지, 이라든지’)

  11.2 시간을 셈할 때 셈에 넣는 한계를 나타내는 말(부사격 조사 ‘로, 으로, 로써, 으로써’)

 ex4) 예술 + 커녕 + 숙제 + 숙제 + 아니다 = 예술은커녕 숙제도 안 한다.
 21. 마찬가지 ‧ 예외
21.1 ‘예외 없이 다 마찬가지로’의 뜻을 나타내는 말(보조사 ‘라고, 이라고’)
 ex1) 여기 + 어디 + 할부로 + 떠들다 + 묻다 = 여기가 어디라고 함부로 떠드느냐?
 ex2) 무엇 + 대접하다 + 없다 + 때문이다 = 무엇이라고 대접할 것이 없어서.
 21.2 다른 경우들과 마찬가지임을 나타내는 말(보조사 ‘라도, 이라도’)
 ex1) 가볍다 + 때문이다 + 어린이 + 도 + 듣다 + 할 수 있다 = 가벼워서 어린이라도 들 수 있다.
 ex2) 아무리 + 선생님 + 도 + 모르다 + 있다 = 아무리 선생님이라도 모르는 것이 있다.
 21.3 ‘예외 없이 모두’의 뜻을 나타내는 말(보조사 ‘치고, 치고는’)
 ex1) 한국인 + 중(中) + 김치 + 모르다 + 사람 + 없다 = 한국인치고 김치를 모르는 사람은 없다.
 ex2) 새끼치고는 + 예쁘다 + 아니다 + 것 + 없다 = 새끼치고는 예쁘지 않은 것이 없다.
 21.4 ‘그 중에서는 예외적으로’의 뜻을 나타내는 말(보조사 ‘치고, 치고는’)
 ex1) 겨울 + 날씨 + 그러나 + 따뜻하다 = 겨울 날씨치고 따뜻하다.
 ex2) 서양 + 사람 + 그러나 + 키가 + 작다 = 서양 사람치고는 키가 작다.
 21.5 ‘치고’를 강조하여 이르는 말(보조사 ‘치고서’)
 ex1) 우리 + 학교 + 학생 + 중(中) + 그 + 모르다 + 사람 + 없다 = 우리 학교 학생치고서 그를 모
르는 사람은 없다.
 21.6 예외성이나 의외성을 강조하는 데 쓰이는 말(보조사 ‘도’)
 ex1) 너 + 신문 + 도 + 읽다 + 아니다/묻는 표정 = 너는 신문도 안 읽니?
 21.7 ‘그것 말고는’, ‘그것 이외에는’의 뜻을 나타내는 말(보조사 ‘밖에’)
 ex1) 나 + 알아주다 + 사람 + 너 + 밖에 + 없다 = 나를 알아주는 사람은 너밖에 없다.
 22. 달리 분류되지 않은 말 
22.1 ‘...로 말하면’, ‘...를 보자면’, ‘...를 들자면’ 따위의 뜻을 나타내는 말(보조사 ‘ㄴ즉, ㄴ즉슨, 인
즉, 인즉슨’)
 ex1) 취지 + 분명하다 = 취진즉 분명하다.
 ex2) 이야기 + 이렇다 = 이야기인즉슨 이렇다.
 ex3) 사실 + 말 + 안된다 = 사실인즉 말이 안 된다.
 ex4) 말씀 + 이렇다 = 말씀인즉슨 이렇다.
 22.2 서로 비슷하거나 같음을 나타내는 말(격 조사 ‘같이, 처럼’)
 ex1) 눈 + 같다 + 희다 + 빛 + 끝 = 눈같이 희다.
 ex2) 동태 + 처럼 + 얼었다 = 동태처럼 얼었다.
 22.3 앞의 말에 근거하여 달라짐이 없음을 나타내는 말(보조사 ‘대로’)
 ex1) 너 + 뜻 + 대로 + 해 = 네 뜻대로 해라.
 22.4 따로따로 구별됨을 나타내는 말(보조사 ‘대로’)
 ex1) 크다 + 것 + 크다 + 것 + 대로 + 따로 + 모아 + 두다 = 큰 것은 큰 것대로 따로 모아 둔다.
 22.5 약속이나 결정을 나타내는 말(격 조사 ‘로’)
 ex1) 마당 + 화초 + 심다 + 결심하다 + 끝 = 마당에 화초를 심기로 결심했다.
 22.6 ‘빠짐없이 모두’를 나타내는 말(보조사 ‘마다’) 
ex1) 사람 + 마다 + 성격 + 다르다 = 사람마다 성격이 다르다.
 22.7 ‘말할 때로’, ‘말할 바와 같이’ 따위의 뜻을 나타내는 말(격 조사 ‘마따나’) 
ex1) 자네 + 말 + 대로 + 극(極) + 통하다 + 법(法) = 자네 말마따나 극과 극은 통하는 법이지.
 22.8 습관적으로 나타나는 현상을 나타내는 말(보조사 ‘만’) 
ex1) 밥 + 만 + 먹다 + -면 + 속이 쓰리다 = 밥만 먹으면 속이 쓰리다.
 22.9 ‘그것만이고 더는 없음’ 또는 ‘오직 그렇게 하거나 그러하다는 것’을 나타내는 말(보조사 ‘뿐’) 
ex1) 믿다 + 것 + 실력 + 뿐 = 믿을 것은 실력뿐이다.
 22.10 앞말이 조건, 환경, 상태 따위의 부사어임을 나타내는 말(격 조사 ‘예’) 
ex1) 이것 + 무더위 + 때 + 지내다 = 이 무더위에 어떻게 지냈니?
 22.11 앞말이 말아보는 자리나 노릇의 부사어임을 나타내는 말(격 조사 ‘예’) 
ex1) 반장 + 그 + 뽑히다 + 끝 = 반장에 그가 뽑혔다.
 22.12 앞말이 근거의 뜻을 갖는 부사어임을 나타내는 말(격 조사 ‘에서’) 
ex1) 고맙다 + 마음 + 가지다 + 드리다 + 말씀 = 고마운 마음에서 드리는 말씀입니다.
 22.13 ‘말할 것도 없거니와 도리어’의 뜻을 나타내는 말(보조사 ‘커녕’) 
ex1) 상(賞) + 커녕 + 벌(罰) + 받다 + 끝 = 상커녕 벌을 받았다.
 22.14 앞 체언이 관형사 구실을 하게 하는 말(관형격 조사 ‘의’) 
ex1) 우리 + 의 + 학교 = 우리의 학교
ex2) 나라 + 의 + 발전 = 나라의 발전
ex3) 나 + 의 + 작품 = 나의 작품
ex4) 승리 + 의 + 길 = 승리의 길
ex5) 자연 + 의 + 관찰 = 자연의 관찰
ex6) 한국 + 의 + 지도 = 한국의 지도
ex7) 책(冊) + 의 + 저자 = 책의 저자

한국수화의 특성
김만영 외(2009)는 한국수화의 특성을 여섯 가지로 설명한다.

주동문 구사
한국수화는 공간을 이용해 주동문을 구사한다. 한국어의 접사(예: '이, 히, 리, 기')를 통해 사동이나 피동을 표현하는 반면, 한국수화는 이러한 형태소가 없고, 공간을 통해 행위자와 행위의 대상을 나타내며, 대부분의 문장이 주어가 직접 행위를 하는 주동문의 형태를 갖는다.

비수지 기호의 문법적 역할
한국어는 어미 활용을 통해 평서문, 의문문 등을 만든다. 그러나 한국수화는 비수지 기호를 통해 문장을 실현한다. 예를 들어, [가다]라는 수화를 하면서 눈썹을 올리고 고개를 앞으로 내미는 비수지 기호는 '가니?'라는 의문문으로 변환된다.

역할 전환(role shift)
여러 주어가 등장할 경우, 한국수화는 역할 전환을 통해 주어를 나타낸다. 예를 들어, 몸을 약간 왼쪽으로 틀면 주어1, 오른쪽으로 틀면 주어2가 된다. 이는 주동문을 효과적으로 구사하기 위한 장치이다.

어순의 제약
한국어는 교착어로 어순이 비교적 자유롭지만, 한국수화는 비굴절 동사의 경우 어순이 중요하다. 비수지 기호와 문법 표지가 없는 비굴절 동사에서 앞에 오는 것이 주어, 나중에 오는 것이 목적어가 된다.

시간 순서에 따른 서술
한국수화 문장은 시간의 순서에 따라 서술하며, 주어가 여러 개인 경우, 주어와 관련된 서술어가 가까이에 위치하도록 한다.

시제 표현
한국어는 서술어의 어미 변화를 통해 시제를 나타내는 반면, 한국수화는 시간을 나타내는 어휘를 통해 시제를 실현하거나, 수화자의 몸을 중심으로 과거-현재-미래를 연속적으로 표현한다.

주어의 실현
한국어는 조사를 통해, 영어는 어순을 통해 서술어의 주어를 나타내지만, 한국수화에서 주어를 나타내는 방법은 서술어의 종류에 따라 다르다.

일반 동사
일반 동사인 경우, 한국어와 달리 조사가 없으므로 주어를 서술어 앞에 놓아 나타낸다. 서술어가 여러 개이거나 복문인 경우는 주어를 일정한 공간에서 해보인 후에, 그 공간 쪽으로 몸을 돌리거나 시선을 주며 서술어의 수화를 한다.

분류사 동사
분류사 동사인 경우, 동사의 손 모양이 주어를 나타낸다. 예를 들어, "꽃이 핀다"는 [피다]라는 동사가 주어를 나타낸다.

방향 동사
방향 동사의 경우, 동사의 출발점이 주어가 된다. 예를 들어, "아빠가 엄마에게 꽃을 주다"라는 문장에서 아빠와 엄마의 위치를 가리키고 동사의 움직임을 시작한다.

목적어의 실현
한국수화에서 목적어를 나타내는 방법은 동사에 따라 다르다.

일반 동사
주어-목적어-서술어의 어순에 의해 나타난다. 목적어가 여러 개이거나 복문인 경우는 목적어를 일정한 공간에서 나타내고 그쪽으로 몸을 돌리거나 시선을 주며 서술어의 수화를 한다.

분류사 동사
분류사 동사인 경우, 동사의 수형을 통해 목적어를 나타낸다. 예를 들어, [국수 먹다]와 [빵 먹다]는 [먹다]라는 수형에 의해 목적어가 나타난다.

일치 동사
일치 동사의 경우, 동사의 도착점이 목적어가 된다. 양 손 수화인 경우 비우세손이 목적어의 역할을 하며 우세손의 방향에 의해 목적어를 나타낸다.

관형어·부사어의 실현
한국어에서는 관형어가 꾸밈을 받는 체언 앞에 놓이지만, 한국수화에서는 서술적 표현으로 이루어지며 두 개의 문장으로 나타난다. 예를 들어, "아이가 새 옷을 입다"는 [아이] [옷] [입다], [옷] [새롭다]라는 두 절로 나뉘어 나타난다.

부사어의 경우는 비수지 기호를 통해 나타내거나, 어휘를 통해 명시적으로 나타낼 수 있다. 예를 들어 "철수가 빨리 달린다"는 [달리다]의 속도나 얼굴 표정을 통해 나타내거나, [달리다] [빠르다]로 표현할 수 있다.

부정 표현
한국수화에서 부정문은 부정을 나타내는 수화 어휘를 사용하거나 비수지 기호를 사용하여 실현할 수 있다. 엄미숙(1996)은 여러 가지 부정법을 분류했다.

사실 진위 부정
동작하는 주체의 의지와 관계없이 사용된다. 예: "그는 학생이 아니다" → [그] [학생] [아니다].

능력 부정
능력이나 외적 원인으로 인해 일이 일어나지 못할 때 사용된다. 예: "나는 수화를 못해요" → [나] [수화] [못하다].

상태나 존재 부정
의미가 없을 때 사용된다. 예: "동생은 예쁘지 않다" → [동생] [예쁘다] [없다].

완료 부정
동작의 완결을 부정할 때 사용된다. 예: "오늘 아침을 안 먹었다" → [오늘] [아침] [먹다] [아직].

기타 부정법
청유문과 명령문에서 사용된다.

방법론적 수화와 한글식 수화
한국에서는 방법론적 수화에 해당하는 수지 한국어가 미국처럼 다양하게 체계적으로 정리되어 있지 않다. 김승국(1994)은 국어 문법에 맞추어 표현할 수 있도록 표준화 작업을 한 뒤 '한글식 수화'라는 개념을 제시하였다. '한글식 수화'는 방법론적 수화인 수지 한국어의 한 형태로, 한국어의 조사와 어미를 나타내는 수화 기호를 자연수화에서 차용하거나 지문자와 수화를 합쳐 만든 기호로 나타낸다.

김승국은 조사를 나타내는 25개의 기호를 제시하고, 나머지 조사는 지문자로 나타내도록 하였다. 예를 들어 '-까지'라는 조사는 [끝]이라는 자연수화 어휘를 그대로 차용하고, '-께'라는 조사는 왼손은 엄지를 펴고 오른손은 지문자 'ㅔ'를 사용해 나타낸다.

어미의 경우도 종결어미 11개, 연결어미 11개, 관형사형 어미 1개, 시제선어말어미 6개 등 총 38개의 기호를 제시하고 나머지 어미는 지문자로 표시하도록 하였다. 예를 들어, '-ㅂ니다'는 손바닥이 위로 향하게 편 왼손바닥에 오른손바닥을 댔다가 떼어 내린다. 의문을 나타내는 '-느냐'는 오른손 검지를 오른쪽 관자놀이에 대었다가 손가락을 모두 펴면서 앞으로 내민다.

하나의 완결된 사상이나 감정을 나타내는 문장 표현은 자연수화의 기호와 지문자를 병용하여 국어 문법에 맞게 표현하도록 하였다. 예를 들어, '아름다운 꽃이 피었다'는 경우 [아름답다] 'ㅜ-ㄴ(지문자)' [꽃] 'ㅣ(지문자)' [피다] [끝] [-다]처럼 자연수화의 기호와 지문자를 사용하여 국어의 모든 문법 정보를 전달한다. 사동문에서는 동사의 수화기호를 주동사의 어간으로 간주해 나타내며, 어미 사이에 [시키다]라는 수화를 삽입한다.
